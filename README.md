## Project Description
Built social backend using Express, MongoDB Atlas, Mongoose and JWT authentication.

- Express
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- Postman

## Features
- User Registration
- Login Authentication
- Protected Routes
- Posts CRUD
- Comments CRUD
- MongoDB Persistence
- Authorization

npm run dev
